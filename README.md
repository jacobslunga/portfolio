# jacobslunga.com

Personal portfolio built with React, TypeScript, Vite, and Tailwind CSS. The site has a home page with a brief introduction, work history, and social links, and an about page with writing, projects, a 3D places carousel, and recent music.

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- MUI Icons
- React Router v7
- Deployed on Netlify

## Development

```bash
npm install
npm run dev
```

## Deployment

The `public/_redirects` file handles SPA routing on Netlify. Push to the connected branch and Netlify builds automatically via `npm run build`.
